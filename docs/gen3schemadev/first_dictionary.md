## Usage

### Installation
```
pip install gen3schemadev
```


### Setup
```
gen3schemadev init
```

### Generate schema
```
gen3schemadev generate -i model.csv -o schema.json
```

### Validate schema
```
gen3schemadev validate -i schema.json
```

### Visualize schema
```
gen3schemadev visualize -i schema.json
```
